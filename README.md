## Problem
Students often prepare well but experience mental blank-outs during high-stress situations.  
This is not a learning failure, but a **stress-induced memory retrieval failure**.

Existing educational tools focus on:
- More content
- Longer explanations
- Repeated revision

These approaches fail during panic.

## Key Insight
The human brain does not fail because information is missing â€”  
it fails because **retrieval collapses under stress**.

Under pressure, the brain can process only:
- Ultra-short cues
- Familiar anchors
- Minimal logical entry points

## Solution Overview
